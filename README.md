WasteSort-AI : Image Based Classification of recyclable and non-recyclable waste

ABSTRACT:
Waste management is a critical global challenge, especially with increasing urbanization and consumption. Improper segregation of waste leads to environmental pollution and inefficient recycling processes. This project presents an intelligent system that automatically classifies waste into recyclable and non-recyclable categories using image-based machine learning techniques. By leveraging deep learning models trained on labeled waste images, the system aims to assist in accurate waste segregation and promote sustainable environmental practices.

PROBLEM STATEMENT:
Manual waste segregation is inefficient, time-consuming, and prone to human error. There is a need for an automated and reliable system that can accurately classify waste materials to support recycling processes and reduce environmental impact.

OBJECTIVE:
-To develop an AI-based system for waste classification
-To automate the process of waste segregation
-To improve recycling efficiency
-To reduce environmental pollution

DATASET:
Source: Kaggle Waste Classification Dataset
Categories: Organic and Recyclable waste
Format: Image dataset
Used for training and testing the model for this project.

METHODOLOGY:
Data Collection – Collected dataset from Kaggle
Data Preprocessing – Image resizing, normalization
Model Development – Training a deep learning model
Model Evaluation – Testing accuracy and performance
Prediction – Classifying new waste images

TOOLS & TECHNOLOGIES:
Python
TensorFlow / Keras
NumPy
Pandas
Matplotlib
Streamlit (for user interface)

RESULT:
The model successfully classifies waste images
Achieved good accuracy on test data
Demonstrates potential for real-world waste management applications

APPLICATION:
Smart waste management systems
Recycling automation
Environmental monitoring
Smart city solutions
